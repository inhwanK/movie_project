## 4. 핵심 기능

### 유저
- 영화 예매
- 영화 조회, 한줄평 및 별점 등록
- 회원가입 및 로그인
- 마이페이지 
- 1:1 문의 

### 관리자
- 공지사항 및 1:1 문의 관리 (CRUD)
- 영화 및 상영정보 관리 (CRUD)
